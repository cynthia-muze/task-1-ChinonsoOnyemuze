DecodeLabs Data Analytics Task 1: Data Cleaning & Preparation

Project Overview

This project involved cleaning and preparing a raw transactional dataset as part of the DecodeLabs Data Analytics Internship Program.
The objective was to improve data quality by identifying missing values, handling inconsistencies, removing duplicates, and correcting data formats.

Tools Used

1. Microsoft Excel
2. Microsoft Power BI (Power Query)

Dataset Summary

1. Total Records: 1,200
2. Total Columns: 14

Data Cleaning Tasks Performed:

Missing Values

1. Identified missing values in the CouponCode column.
2. Replaced missing CouponCode values with "NO COUPON".

Duplicate Checks

1. Checked OrderID for duplicate records.
2. Verified that no duplicate OrderIDs existed.

Data Formatting

1. Standardized date fields.
2. Validated numeric columns.
3. Standardized text formatting.
4. Applied Trim and Clean transformations where required.

Validation

1. Verified TotalPrice calculations.
2. Checked OrderID consistency.
3. Checked TrackingNumber consistency.

Final Outcome

1. Zero duplicate IDs
2. Zero incorrectly formatted dates
3. Missing values handled appropriately
4. Dataset prepared for analysis and reporting
